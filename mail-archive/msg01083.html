Hello, members of the Italian team at `it@li.org'.  This is a message
from the Translation Project robot.  I`m happy to announce that a new
file, available as:

>    http://www.iro.umontreal.ca/contrib/po/trans/it/findutils-4.1.5.it.po

has been integrated in the central PO archives, and is now kept with
all other accepted Italian translations.  The file should soon be made
available in mirror sites as:

>    ftp://ftp.unex.es/pub/gnu-i18n/po/trans/it/findutils-4.1.5.it.po
>    ftp://ftp.chg.ru/pub/doc/gnu-i18n//trans/it/findutils-4.1.5.it.po

All its 95 messages have been translated, and this PO file has been
submitted to the maintainer of `findutils', hoping s/he will include
it in a future release of programs using this textual domain.

Let me thank you for all users of the Italian language.

The following HTML pages should also be updated by tomorrow.

>    http://www.iro.umontreal.ca/contrib/po/HTML/domain-findutils.html
>    http://www.iro.umontreal.ca/contrib/po/HTML/team-it.html

                                The Translation Project robot, in the
                                name of your kind translation coordinator.
                                mailto:translation@iro.umontreal.ca
